## 2. Advertising (Google AdMob)

The App uses **Google AdMob** to display advertisements.

AdMob may automatically collect and process data such as:

- Device information (device type, OS version)
- Approximate location (for ad relevance)
- App usage and interaction with ads
- Advertising identifiers (IDFA / AAID), **if permission is granted**

This data is handled and stored by **Google**, not by the App developer.

Google’s policy:  
https://policies.google.com/technologies/ads

## 3. Consent & User Control

### **GDPR (EEA users)**  
Users in the European Economic Area will be asked if they want:

- Personalized ads **or**
- Non-personalized ads

Consent can be changed at any time in the App’s privacy settings.

### **iOS Tracking (ATT)**  
Apple may show a system prompt about allowing tracking across other apps.

- If allowed → personalized ads may be shown
- If denied → only **non-personalized ads** will be shown

The App continues to function normally in both cases.
